# KlauzDB [![NPM version](https://img.shields.io/npm/v/dotenv.svg?style=flat-square)](https://www.npmjs.com/package/klauz-db)

KlauzDB é um banco de dados orientando a "collections", que utiliza o armazenamento local da máquina para persistir os dados, seguindo sua proposta de ser rápido e de fácil utilização, não requerindo nenhum servidor externo ou afins. 

Ideal para testes automatizados, vídeo-aulas ou aplicações com um grau menor de complexidade.
